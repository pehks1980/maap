# maap

Тренажер арифметики для детей начальных классов.

Тренирует и проверяет основные арифметические операции: сложение/ вычитание/ таблица/ умножения/ деление.

Тренажер часов: просит сказать время на часах, время на часах через n час и n минут.

Отсуствует ненужные детские анимации (не релевантно, т.к. основная задача - научить соображать ребенка)

Родители могут смотреть отчеты о занятиях, и анализировать трудные примеры, время их решения.

Ребенок должен решать задачи в уме.

Ребенок может лениться сначала, но при регулярных занятиях на тренажере, замечена "положительная динамика" - ребенок быстрее решает примеры, 
делает меньше ошибок, в школе дела с математикой улучшаются. Главное - регулярные занятия.
